# Casa Blu Marbella Website

Ez a projekt egy egyszerű, statikus weboldal, amely bemutatja a Casa Blu Marbella luxus nyaralót.

## 🚀 Használat

1. Töltsd le a teljes ZIP-et.
2. Bontsd ki a fájlokat.
3. Nyisd meg a terminált/mappát a projekttel.
4. Futtasd az alábbi parancsokat:

```bash
npm install
npm start
